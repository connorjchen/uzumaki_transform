## How to Get Setup

1. Install pyenv

   1. brew install pyenv
   2. nano ~/.zshrc
   3. Copy paste at bottom of .zshrc
      ```
      export PYENV_ROOT="$HOME/.pyenv"
      command -v pyenv >/dev/null || export PATH="$PYENV_ROOT/bin:$PATH"
      eval "$(pyenv init -)"
      ```
   4. Reload terminal

2. pyenv install 3.11.0

3. sh setup.sh

## How to Run

1. sh run_main.sh
